# Advent of Code (in Rust)

No peeking! (unless you've already done it)

Trying to solve Advent of Code puzzles in Rust, while learning Rust.

## Usage

```
cargo run 2022 1
```
or, once built:
```
aoc 2022 1
```

## Build

```
cargo build --release
```
